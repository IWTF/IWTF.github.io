Hexo-Blog-Encrypt实现私密文章

以下内容来自该插件的中文说明文档，详见github。

安装
在 hexo 根目录的 package.json 中添加 “hexo-blog-encrypt”: “1.1.*” 依赖。
然后执行npm install 命令。
该插件会自动安装

快速开始

首先在 _config.yml 中启用该插件:

在你的文章的头部添加上对应的字段，如 password, abstract, message
